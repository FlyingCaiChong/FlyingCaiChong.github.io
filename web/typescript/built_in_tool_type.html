# 内置工具类型

# Partial<T>

作用

将一个类型的属性 全部变为可选属性

定义:

/**
 * Make all properties in T optional
 */
type Partial<T> = {
  [P in keyof T]?: T[P];
};
1
2
3
4
5
6

该 Type 使用时需要传入一个泛型 T, 内部遍历 T 的所有属性, 然后创建一个新的 Type, 新的 Type 的所有属性使用 ? 标识, 使之为可选.

keyof 会遍历一个 Interface 的所有属性名称(key), 生成一个联合类型.

示例:

interface A {
  x: number;
  y: number;
}
// Partial<A>: { x?: number; y?: number }
const a: Partial<A> = {
  x: 1,
};

# Required<T>

作用

将一个类型的属性 全部变为必选属性

定义:

/**
 * Make all properties in T required
 */
type Required<T> = {
  [P in keyof T]-?: T[P];
};
1
2
3
4
5
6

该 Type 和 Partial 刚好相反, 使用时需要传入一个泛型 T, 内部使用 -? 将 T 的每个属性去除可选标识, 使之为必选.

示例:

interface A {
  x?: number;
  y: number;
}
// Required<A>: { x: number; y: number }
const a: Required<A> = {
  x: 1,
  y: 1,
};

# Readonly<T>

作用

将一个类型的属性 全部变为只读属性

定义:

/**
 * Make all properties in T readonly
 */
type Readonly<T> = {
  readonly [P in keyof T]: T[P];
};
1
2
3
4
5
6

该 Type 使用时需要传入一个泛型 T, 内部使用 readonly 将 T 的每个属性去除可选标识, 使之为只读.

示例:

interface A {
  x?: number;
  y: number;
}
// ReadOnly<A>: { readonly x: number; readonly y: number }
const a: ReadOnly<A> = {
  x: 1,
  y: 1,
};
a.x = 2; // error

# Record<K, T>

作用

使用给定的对象属性名类型和对象属性类型创建一个新的对象类型

定义:

/**
 * Construct a type with a set of properties K of type T
 */
type Record<K extends keyof any, T> = {
  [P in K]: T;
};

示例:

type K = "x" | "y";
type T = number;
type R = Record<K, T>; // {x: number, y: number}

# Pick<T, K>

作用

从一个Type中选取一些属性, 来构造一个新的对象Type

定义:

/**
 * From T, pick a set of properties whose keys are in the union K
 */
type Pick<T, K extends keyof T> = {
  [P in K]: T[P];
};
1
2
3
4
5
6

类型参数T表示源对象类型，类型参数K提供了待选取的属性名类型，它必须为对象类型T中存在的属性

示例

interface A {
  x: number;
  y: number;
}
type T0 = Pick<A, "x">; // { x: number }
type T1 = Pick<A, "y">; // { y: number }

# Omit<T, K>

作用

从一个对象类型中删除一些属性来构造一个新的对象Type

定义

/**
 * Construct a type with the properties of T except for those in type K.
 */
type Omit<T, K extends keyof any> = Pick<T, Exclude<keyof T, K>>;
1
2
3
4

和Pick刚刚相反, 用于排除不需要的属性. 类型参数T表示源对象类型，类型参数K提供了待剔除的属性名类型，但它可以为对象类型T中不存在的属性.

示例

interface A {
  x: number;
  y: number;
}
type T0 = Omit<A, "x">; // { y: number }
type T1 = Omit<A, "y">; // { x: number }
type T2 = Omit<A, "z">; // { x: number; y: number }

# Exclude<T, U>

作用

排除一个联合类型中的某一些类型, 来构造一个新的Type

定义

/**
 * Exclude from T those types that are assignable to U
 */
type Exclude<T, U> = T extends U ? never : T;
1
2
3
4

示例

type T0 = Exclude<"a" | "b" | "c", "a">; // 'b' | 'c'

# Parameters<T>

作用

从[函数Type]的形参中, 构造一个数组Type

定义

/**
 * Obtain the parameters of a function type in a tuple
 */
type Parameters<T extends (...args: any) => any> = T extends (
  ...args: infer P
) => any
  ? P
  : never;
1
2
3
4
5
6
7
8

infer 标识一个待推导类型, 上面定义的意思是: 如果 T 为函数类型, 那就返回函数的形参;

infer 和变量似的，先定义一个 infer P 然后 Ts 就会自动推导函数的形参或者返回值、或者数组元素等，然后开发者在合适的位置使用定义好的infer P即可。

infer案例: 需求: 需要将数组类型的 Type 变为联合类型。其他类型的则不变.

type ArrayToUnion<T> = T extends Array<infer Item> ? Item : T;
const a: ArrayToUnion<[string, number]> = "111"; // a: string|number
1
2

示例 1:

type T0 = Parameters<() => string>; // []
type T1 = Parameters<(s: string) => void>; // [string]
1
2

示例 2:

function getUserInfo(id: string, group: string) {}
type GetUserInfoArg = Parameters<typeof getUserInfo>;
const arg: GetUserInfoArg = ["001", "002"];
getUserInfo(...arg);
1
2
3
4

上面代码中的typeof是 ts 提供的操作符不是 js 中的那个typeof，只能用到 ts 的类型定义中, 所以使用typeof getUserInfo才能指向函数Type.

# ConstructorParameters<T>

作用

从定义的[构造函数]的形参中, 构造一个数组Type

定义

/**
 * Obtain the parameters of a constructor function type in a tuple
 */
type ConstructorParameters<T extends new (...args: any) => any> =
  T extends new (...args: infer P) => any ? P : never;
1
2
3
4
5

实现原理完全和 Parameters 一样，只不过这个方法接受的是一个类。

示例

class User {
  constructor(id: string, group: string) {}
}
type NewUserArg = ConstructorParameters<typeof User>;
const arg: NewUserArg = ["001", "002"];
new User(...arg);

# ThisParameterType<T>

作用

提取函数Type的 this 参数, 生成一个新的Type, 若函数类型中没有定义this参数, 则返回unknown类型.

定义

/**
 * Extracts the type of the 'this' parameter of a function type, or 'unknown' if the function type has no 'this' parameter.
 */
type ThisParameterType<T> = T extends (this: infer U, ...args: any[]) => any
  ? U
  : unknown;
1
2
3
4
5
6

示例

function getUserInfo(this: { name: string }) {}
const getUserInfoArgThis: ThisParameterType<typeof getUserInfo> = {
  name: "zhang",
};

# OmitThisParameter<T>

作用

忽略函数Type 的this参数, 生成一个新的函数Type

定义

/**
 * Removes the 'this' parameter from a function type.
 */
type OmitThisParameter<T> = unknown extends ThisParameterType<T>
  ? T
  : T extends (...args: infer A) => infer R
  ? (...args: A) => R
  : T;

4
5
6
7
8

示例

function getUserInfo(this: { name: string }, id: string) {}
const a: OmitThisParameter<typeof getUserInfo> = (id: string) => {};
